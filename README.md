# Store
A simple online clothing and shoe store. Implemented shopping cart, filtering by category, pagination, admin panel, user authorization.

Development stack:
  - Python 3.10
  - Django 3.2.18
  - SQlite 3.37.2
  - Pillow 9.5.0
